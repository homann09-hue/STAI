# Security Overview

Kontrollen: CSP/HSTS, API-Validierung, Body-/Response-Limits, Rate Limits, Same-Origin, serverseitige Keys, Timing-safe Adminauth, RLS, Composite Owner-FKs, Logredaction, append-only Auditmodell und CI-Scans. Keine Aussage ersetzt einen externen Penetrationstest. Matrix: `../security-control-matrix.md`.
